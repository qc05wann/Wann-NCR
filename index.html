วิธีใช้:
1. กด ดาวน์โหลด Template Excel → กรอกข้อมูลจริงลงในแต่ละ Sheet
2. กด Import Excel → เลือกไฟล์ที่กรอกแล้ว
3. ข้อมูลจะถูกบันทึกขึ้น Firebase Cloud โดยอัตโนมัติ
4. กด Export Backup เป็นประจำเพื่อสำรองข้อมูล
